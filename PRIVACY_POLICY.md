# 개인정보처리방침

Naver Store Stock Overlay는 네이버 스마트스토어 및 브랜드스토어 상품 목록 페이지에서 상품별 재고 수량을 확인하기 위한 Chrome 확장 프로그램입니다.

## 처리하는 데이터

이 확장 프로그램은 지원 대상 페이지에서 다음 데이터를 브라우저 안에서만 처리합니다.

- 현재 활성 탭의 URL과 제목: 지원 대상 페이지인지 확인하고 팝업에 상태를 표시하기 위해 사용합니다.
- 네이버 스마트스토어 및 브랜드스토어 페이지의 웹사이트 콘텐츠: 상품 링크, 상품 이미지, 상품명, 상품 ID를 확인하기 위해 사용합니다.
- 페이지 HTML에 포함된 초기 페이지 상태 JSON: 상품 ID와 재고 수량 후보 필드를 감지하기 위해 사용합니다.
- 확장 프로그램 설정값: 오버레이 켜기/끄기 상태를 `chrome.storage.local`에 저장합니다.

## 데이터 사용 목적

처리된 데이터는 상품 이미지 위에 재고 수량 배지를 표시하고, 확장 프로그램 팝업에서 현재 감지 상태를 보여주기 위해서만 사용됩니다.

## 데이터 저장 및 전송

이 확장 프로그램은 감지한 상품 정보, 웹사이트 콘텐츠, 방문 기록을 외부 서버로 전송하지 않습니다. 재고 감지와 오버레이 표시는 사용자의 브라우저 안에서만 수행됩니다.

확장 프로그램이 저장하는 값은 오버레이 켜기/끄기 설정값뿐이며, 이 값은 사용자의 브라우저 로컬 저장소에만 저장됩니다.

## 제3자 제공 및 판매

이 확장 프로그램은 사용자 데이터를 제3자에게 판매하지 않으며, 확장 프로그램의 전용 목적과 관련 없는 목적으로 사용자 데이터를 전송하거나 사용하지 않습니다. 또한 사용자 데이터를 신용도 판단 또는 대출 목적으로 사용하지 않습니다.

## 원격 코드

이 확장 프로그램은 패키지에 포함된 코드만 실행하며, 원격 JavaScript 또는 WebAssembly 코드를 로드하거나 실행하지 않습니다.

## 문의

확장 프로그램의 개인정보 처리에 관한 문의는 Chrome 웹 스토어 개발자 연락처를 통해 문의할 수 있습니다.
